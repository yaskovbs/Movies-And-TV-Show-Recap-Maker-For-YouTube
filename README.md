# Movies And TV Show Recap Maker For YouTube

A web application that helps create recaps of movies and TV shows for YouTube. This tool automates the process of creating engaging recap videos with customizable settings.

## Features

- Upload videos for processing
- Customize recap settings
- Generate voiceovers
- Track processing status
- Save and manage your recaps
- View statistics

## Technologies Used

- React
- TypeScript
- Firebase
- Supabase
- Tailwind CSS
- Vite

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Configure environment variables
4. Run the development server with `npm run dev`
